# 内存

  • 引用数据类型 Object
  • 内存控件：
  • 栈内存 Stack （原始数据类型）

# JS垃圾回收机制：

按照固定的时间间隔周期性执行，找出无用变量 释放内存。

# 优缺点：

  • 开发不需要关注
  • 不可见，无法干预。

# 标记清除：

当变量进入环境时，标记为进入执行环境 每个执行环境都有一个关联的变量对象variable object 例如，函数执行结束，其中的局部变量离开了环境，内存被回收

浏览器中，全局执行环境是window对象
